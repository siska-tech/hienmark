# HienMark 要件定義書

## 概要

本ドキュメントは、ローカルファーストで動作する高性能な「タスク管理エディタ」の開発に関する要件を定義します。

本エディタは、単一のMarkdownファイルを「タスク」という単位として扱い、ファイルに付与された「タグ情報（フロントマター）」を基盤とした強力な管理・分析機能を統合することを特徴とします。

## プロジェクトの目的と背景

現状、多くのMarkdownエディタは「執筆」に焦点を当てており、プロジェクト管理やタスク分析の機能は外部ツールに依存しています。一方で、プロジェクト管理ツールは多機能ですが、クラウド依存であり、ローカルでの軽快なドキュメント作成には不向きな場合が多いです。

本プロジェクトの目的は、これら二つの領域を「ローカルファースト」の思想で統合することにあります。高速で軽量なエディタ体験（執筆）と、EChartsを活用した視覚的なデータ分析（管理）を単一のアプリケーションで完結させ、開発者やライターの生産性を向上させる新しいカテゴリのツールを創出します。
